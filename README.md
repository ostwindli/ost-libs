# ost-tools

Ostwind's JavaScript Tools

## Dev

```sh
# init a new tool
npm run new

# test
cd packages/PKG_NAME
npm run test

# publish tool
npm run pub PKG_NAME
#or
cd packages/PKG_NAME
npm run pub


# generate a tool doc
npm run docs [PKG_NAME]
#or
cd packages/PKG_NAME
npm run docs

# deploy docs
npm run docs:deploy
```
