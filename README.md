# asenal

ostwindli 的武器库

## 本地开发

### 初始化一个新包

```js
npm run new
```

### 生成 API 文档

执行下面命令，自动生成 readme

```sh
#eg: npm run gen-docs string
npm run docs PKG_NAME

```

或者

```sh
cd packages/PKG_NAME
npm run docs
```

### 测试

```sh
cd packages/PKG_NAME
npm run test
```

### 发布

```sh
#eg: npm run gen-docs string
npm run pub PKG_NAME

```

或者

```sh
cd packages/PKG_NAME
npm run pub
```
