# Asenal

Ostwindli's JavaScript Tools

## Local Dev

### Init a new package

```js
npm run new
```

### Generate API Docs

exec below shell

```sh
#eg: npm run docs string
npm run docs PKG_NAME

```

or

```sh
cd packages/PKG_NAME
npm run docs
```

### Test

```sh
cd packages/PKG_NAME
npm run test
```

### Publish

```sh
npm run pub PKG_NAME

```

or

```sh
cd packages/PKG_NAME
npm run pub
```

### Deploy Docs

```sh
npm run docs:sync
```
