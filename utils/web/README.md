# @ostwindli/web

a lot of js tools for browser

## 安装

```sh
$ yarn add @ostwindli/web
```

## 使用方法

```js
const { scrollToTop } = require("@ostwindli/web");
scrollToTop();
```

## 功能清单

  ```ts
  

    ls(name: any, value: any, expires: any): any;

    scrollToTop: () => void;

    getPublicNetworkIp(): Promise<string>;

    contextmenuBan: () => void;



  ```
